# Список дел пользователей

- ### Была реализована авторизация и регистрацией пользователей.

- ### Был применен паттерн DTO для передачи типизированных переменных между слоями приложения.

- ### Была вынесена общая логика валидации у классов LoginRequest и RegistrationRequest в отдельный трейт.

- ### Каждая новая задача привязана к пользователю по его id, реализована проверка прав на удаление задач.
